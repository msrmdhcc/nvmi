# NVMI 

A Open Source - lightweight real-time NVIDIA GPU monitor written in Python.

![NVMI Live GPU Monitor](screenshots/nvmi-display-new.png)

## Features
- Live GPU utilization
- Memory usage
- Temperature
- Power draw
- Single-box live UI (nvidia-smi style)
- Works on Nvidia Optimus

## Requirements
- NVIDIA GPU
- NVIDIA drivers
- Python (for who wants to build)

## Install
```bash
pip install -r requirements.txt
